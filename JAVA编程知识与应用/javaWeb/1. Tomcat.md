# Tomcat

## 网站结构

```java
---webapps  : Tomcat 服务器的目录名
    -ROOT
    -studyWeb: 网站的目录名
        -WEB-INF
        	-classes : java 程序
            -lib:   web应用所依赖的jar包
            -web.xml:  网站配置文件
        -index.html: 默认网站首页
        -static  :网站静态文件
            -css
            	-style.css
            -js
            -img
         -......
```
